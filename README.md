# Лабораторна робота №5. Робота з системою контролю версій Git
**Тема:** Розгалужена робота у системі контролю версій Git
**Опис:** Освоєння роботи з гілками, злиттям та віддаленими репозиторіями
**Мета:** Опанувати практичні навички розгалуженої роботи в Git

## Контрольні питання
1. Для чого потрібна команда git push?
Відправляє локальні коміти у віддалений репозиторій.

2. Що таке конфлікт злиття (merge conflict)?
Ситуація, коли Git не може автоматично об’єднати зміни.

3. Для чого використовується команда git clone?
Для копіювання віддаленого репозиторію на локальний комп’ютер.

4. Що робить команда git add?
Додає зміни до індексу перед комітом.
